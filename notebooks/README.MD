This folder contains notebooks used for analysis, experimentation, and reference.

The main notebooks are:

- `data_understanding.ipynb` — exploratory work for understanding the dataset and early analysis ideas
- `ml.ipynb` — notebook version of the Stage 3 modeling logic

These notebooks are reference material and are not supposed to be the core execution layer during assessment.

The production pipeline lives in `scripts/`, especially in `scripts/modeling.py` for Stage 3.
